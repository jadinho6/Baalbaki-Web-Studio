# PROJECT OVERVIEW

This project is the portfolio and lead-generation website for:

Baalbaki Web Studio

The website represents a modern web design service for Swiss small and medium-sized businesses (KMU).

---

# GOAL

The main goal of this website is to:

* present the business professionally
* build trust with potential clients
* showcase design capability
* generate project inquiries

The website must feel modern, clear, and professional.

Visitors should quickly understand:

* what we do
* who it is for
* why it is valuable

---

# TARGET AUDIENCE

Swiss small and medium-sized businesses (KMU), especially:

* local service businesses
* restaurants
* practices (e.g. physiotherapy)
* small companies

These clients value:

* clarity
* professionalism
* simplicity
* trust

---

# POSITIONING

Baalbaki Web Studio is positioned as:

* modern
* professional
* reliable
* clear and easy to work with

Not:

* cheap
* overly technical
* exaggerated or “marketing-heavy”

---

# COMMUNICATION STYLE

* Language: German (formal “Sie”)
* Tone: clear, calm, professional
* Perspective: always “wir”

Avoid:

* hype language
* buzzwords
* exaggerated promises

---

# CORE PRINCIPLES

* clarity over complexity
* quality over quantity
* structure over decoration
* trust over hype

The website should feel like a professional, well-designed solution — not a template or beginner project.
